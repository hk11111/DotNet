CSS -
Cascading Style Sheet
אחראית על עיצוב העמוד

החלת עיצוב על אלמנט בHTML -
יבוצע בעזרת סימון האלמנט, פתחית סוגריים מסולסלים וסגירת סוגריים מסולסלים.
בין הסוגריים יחול העיצוב הגרפי המבוקש, לדוגמא:

p {
font-family: Tahoma, Geneva, sans-serif;
}

http://www.asmarterwaytolearn.com/htmlcss/4.html

תגית link - מקשרת קובץ CSS לתוך קובץ הHTML
תופיע בתוך תגית HEAD - לדוגמא:

<head>
<link rel="stylesheet" type=text/css" href="style.css">
</head>

תרגיל שני:
נשתמש בעמוד האינטרנט שהכנו בתרגיל 1, הוסיפו לו תגית link בתוך תגית הhead, וקשרו אליו קובץ CSS עם הפרטים הבאים:
שמכם הפרטי יהיה בגודל של 50 פיקסלים, ובפונט "Arial"
שם המשפחה יהיה בגודל 70 פיקסלים, ובפונט "Tahoma"
עיר המגורים תהיה בגודל 30 פיקסלים, ובפונט "Times New Roman"
הפיסקה תהיה בגודל פי 2.2 מהגודל הסטנדרטי ובפונט "Georgia"
לכל הפונטים יש לשים פונט גיבוי - Geneva, ומשפחת פונטים sans-serif.
